TaskManager Java + MySQL

Aplicação Java de gerenciamento de tarefas com conexão ao banco de dados MySQL via JDBC.  
Permite cadastrar, editar, concluir e excluir tarefas.  
Desenvolvido com foco em aprendizado, prática de JDBC, POO e integração com banco relacional.

Tecnologias

- Java 11
- MySQL
- JDBC
- IntelliJ IDEA
- Maven

Funcionalidades

- Login e cadastro de usuário
- Cadastro de tarefas
- Listagem de tarefas
- Edição de tarefas
- Exclusão de tarefas
- Conclusão de tarefas
- Desconclusão de tarefas
- Integração com banco de dados MySQL

Banco de Dados

O arquivo `databasetaskmanager.sql` contém o script para criação do banco e tabelas.  
Importe esse script no seu MySQL antes de executar o projeto.

Estrutura básica esperada:

- Banco: `taskmanager`
- Tabela: `tarefas`
